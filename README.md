# KonselingKu

Website Konseling Online berbasis PHP dan MySQL yang dirancang untuk mempermudah proses konseling antara siswa dan guru BK secara digital.

## Fitur

### Siswa
- Login & Register
- Dashboard
- Profil
- Ajukan Konseling
- Melihat Jadwal Konseling
- Chat dengan Guru BK
- Riwayat Konseling

### Guru BK
- Login
- Dashboard
- Profil
- Data Siswa
- Kelola Konseling
- Membuat Jadwal Konseling
- Chat dengan Siswa

## Teknologi

- PHP Native
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## Struktur Project

```plaintext
konseling-online/
├── assets/
├── config/
├── templates/
├── siswa/
├── guru/
└── database/
```

## Cara Menjalankan

1. Clone repository

```bash
git clone URL_REPOSITORY
```

2. Import database

```plaintext
database/konseling.sql
```

3. Jalankan XAMPP

- Apache → Start
- MySQL → Start

4. Buka browser

```plaintext
http://localhost/konseling-online
```
